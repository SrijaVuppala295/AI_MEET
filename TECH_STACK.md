# Tech Stack of the Project

This document summarizes the complete technology stack used in the AI Interview Agent project.

---

## 1. Frontend

| Category | Technology |
|---|---|
| Framework | Next.js 15.5 |
| UI Library | React 19 |
| Language | TypeScript |
| Styling | Tailwind CSS |
| Component Library | shadcn/ui, Radix UI |
| Icons | Lucide React |
| Theme Support | next-themes |
| Markdown Rendering | react-markdown |
| Forms | React Hook Form |
| Validation | Zod |

---

## 2. Backend

| Category | Technology |
|---|---|
| Server Framework | Next.js API Routes |
| Server Language | TypeScript |
| Authentication | Firebase Authentication |
| Server-side Firebase | Firebase Admin SDK |
| Database | Firestore |
| Server Actions | Next.js Server Actions |

---

## 3. AI and Voice Services

| Category | Technology |
|---|---|
| AI Model Provider | Groq |
| AI Model Provider | OpenRouter |
| AI Model Provider | Google Generative AI |
| Voice AI | Vapi AI |
| Speech-to-Text / Voice Pipeline | Vapi + Voice Web SDK |

---

## 4. Data and File Handling

| Category | Technology |
|---|---|
| PDF Parsing | pdf-parse-fork |
| DOCX Parsing | mammoth |
| JSON Data | Static JSON files |
| Unique IDs | uuid |
| Date Handling | dayjs |

---

## 5. UI/UX and Visualization

| Category | Technology |
|---|---|
| Charts | Recharts |
| Toast Notifications | Sonner |
| Utility Styling | clsx, tailwind-merge |
| Animation | tailwindcss-animate |

---

## 6. Development Tools

| Category | Technology |
|---|---|
| Package Manager | npm |
| Linter | ESLint |
| Type Checking | TypeScript |
| Build Tool | Next.js Build |
| Deployment | Netlify |

---

## 7. Summary
The project uses a modern full-stack tech stack centered around:
- Next.js for the application framework,
- React and TypeScript for the frontend,
- Firebase for authentication and database,
- AI services for chatbot, quiz generation, feedback, and prep analysis,
- and Vapi for voice-based interview simulation.
