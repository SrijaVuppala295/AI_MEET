## 🚀 System Overview

AI MEET is a modern **Next.js 15** application designed for maximum performance and user engagement. It bridges the gap between static study and real-world interview pressure through:

1.  **Voice-First Interaction**: Using Vapi for natural conversation.
2.  **AI-Native Features**: Leveraging Gemini and Llama for deep analysis.
3.  **Modern UI/UX**: Built with Framer Motion, Recharts, and Tailwind CSS for a premium feel.
4.  **Secure Foundation**: Firebase-backed security and Firestore data persistence.
